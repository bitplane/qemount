# Linux 2.6 Compiler

Ubuntu 10.04 (Lucid) based compiler with GCC 4.4. Required for building
Linux 2.6.39 kernel which is incompatible with modern GCC.

Only x86_64 is supported - Ubuntu 10.04 Docker images are only available
for amd64 architecture.
