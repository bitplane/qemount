# TIM-011 disk image

A raw, headerless 3.5" floppy sector image for the TIM-011, an educational
computer used in Serbian/Yugoslav schools from 1987 into the 1990s (around 1200
units, mostly in Belgrade). The TIM-011 ran CP/M on a Hitachi HD64180 — a
Z80-compatible CPU with an on-chip MMU — with 256 KB of RAM and an integrated
512x256 green-screen monitor.

The image is a flat dump of disk sectors with no header or magic. MAME's loader
maps the byte stream onto a single fixed geometry through a UPD765-class
controller model; the source notes its gap sizes are unverified. Images are
identified by size, the `.img` extension and context, not by a signature.

## Geometry

| Property | Value |
|----------|-------|
| Tracks (cylinders) | 80 |
| Heads | 2 |
| Sectors/track | 5 |
| Bytes/sector | 1024 |
| Encoding | MFM (double-sided, double density) |
| Data rate | 250 kbit/s |
| Total | 800 KB |
