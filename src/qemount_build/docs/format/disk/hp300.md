# HP 9000/300 floppy image

This is a raw sector image of a floppy disk as used by Hewlett-Packard's
HP 9000 Series 300 workstations (Motorola 68000-family machines running HP-UX
or BASIC/Pascal workstation systems, mid-1980s onward). The Series 300 used
HP-IB connected flexible-disc drives such as the HP 9121, 9122, and 9133, and
its discs typically carry an HP LIF volume (`format/fs/hp-lif`).

## Structure

The file is a flat, headerless dump of every sector in cylinder/head/sector
order — there is no magic number or container, so the image is identified by its
geometry and context rather than by any signature. MAME's loader recognises a
family of HP 9121/9122/9133 geometries, all MFM-encoded with 256/512/1024-byte
sectors:

- 3.5" single-sided DD: 16 sectors/track, 66 tracks, 256-byte sectors.
- Double-sided DD variants: 16×77×256, 9×77×512, 5×77×1024, and 9×80×512.
- High-density variants with 32, 18, or 10 sectors per track at the matching
  sector sizes.

Because there is no header, no signature-based detection is defined here; the
on-disk filesystem is normally HP LIF.
