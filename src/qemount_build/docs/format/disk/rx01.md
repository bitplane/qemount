The RX01 is Digital Equipment Corporation's first 8-inch floppy disk
subsystem, introduced in the mid-1970s for the PDP-8 (as the RX8/RX8E)
and PDP-11 (as the RX11). An RX01 image is a raw sector dump of one of
these single-sided 8-inch diskettes.

The geometry follows the IBM 3740 single-sided single-density standard:
77 tracks, 1 head, 26 sectors per track of 128 bytes each, recorded in
FM (single density). That gives 77 x 26 x 128 = 256,256 bytes per disk.

MAME's loader ("RX01 floppy disk image") carries no header or magic; it
identifies an image purely by matching its length against the expected
77 x 26 x 128 size and then applies the basic fixed-disk geometry. As a
headerless raw dump there is nothing to detect beyond that size, so no
signature-based detection is documented here.

The successor RX02 doubled capacity using a DEC-specific
double-density MFM scheme on the same media; this loader covers the
original single-density RX01 only.
