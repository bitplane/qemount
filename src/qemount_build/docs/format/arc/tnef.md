# TNEF (Transport Neutral Encapsulation Format)

TNEF is Microsoft's format for encoding email message properties and
attachments, used by Exchange Server and Outlook. When a richly-formatted
email is sent to a non-Microsoft recipient, the TNEF data appears as the
infamous `winmail.dat` attachment that nobody can open.

## Characteristics

- Encodes rich text formatting, embedded objects, and attachments
- Generated by Microsoft Exchange and Outlook
- Appears as `winmail.dat` or `ATT00001.dat` to non-MS clients
- Binary format with nested attribute/value structure
- Contains original message properties and attached files

## Structure

```
Header:
  Offset  Size  Field
  0       4     Magic (0x223E9F78, little-endian)
  4       2     API key (attachment key)
```

Followed by a sequence of attribute records, each containing:
- Level byte (message-level or attachment-level)
- Attribute ID (2 bytes)
- Data length (4 bytes)
- Data
- Checksum (2 bytes)

## File Extension

`winmail.dat`, `.tnef`
