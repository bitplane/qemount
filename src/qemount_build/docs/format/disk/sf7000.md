# SF-7000 disk image

A raw, headerless sector image for the **Sega SF-7000 Super Control Station**,
the disk-drive expansion unit for the Sega SC-3000 home computer (1983). The
SF-7000 adds a 3" compact floppy drive plus extra RAM and ROM; this `.sf7`
image is a flat dump of one disk's sectors, which MAME reads via its generic
uPD765 floppy-controller image handling with the SF-7000's fixed geometry.

## Geometry

Fixed geometry, no header:

| Property | Value |
|----------|-------|
| Tracks | 40 (numbered 0-39) |
| Heads / sides | 1 (single-sided) |
| Sectors / track | 16 |
| Bytes / sector | 256 |
| Encoding | MFM (double density) |
| Total size | 163,840 bytes (160 KB) per side |

Because the image has no container header or magic bytes, it is identified by
its `.sf7` extension and its fixed 163,840-byte size rather than by a signature,
so no detection rule is given here. The SC-3000's cassette counterpart is
catalogued separately at [media/sc3000-bit](../media/sc3000-bit.md).
