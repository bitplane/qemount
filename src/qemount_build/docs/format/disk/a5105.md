# A5105 (Robotron floppy image)

A raw, headerless sector image of a floppy for the Robotron A5105 — also sold as
the BIC (*Bildungscomputer*, "education computer") — a Z80 machine built by VEB
Robotron in East Germany in 1989–1990, largely for schools. The A5105 ran CP/M
from its floppy-controller ROM, and an 800 KB disk image is a straight dump of
its sectors.

## Geometry

Fixed geometry, MFM encoding:

| Property | Value |
|----------|-------|
| Tracks | 80 |
| Sides | 2 |
| Sectors / track | 5 |
| Bytes / sector | 1024 |
| Total | 819,200 bytes (800 KB) |

There is no header or signature; the image is recognised by its size and
geometry. MAME notes that its gap sizes are unverified.
