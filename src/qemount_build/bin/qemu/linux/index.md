# Linux QEMU Guests

Linux-based virtual machines for QEMU. Each version provides different
filesystem support - use 2.6 for legacy formats, 6.12 for broad modern
coverage.

## Filesystem support history

History of in-tree filesystem support across Linux kernel releases. Only
covers additions and removals of filesystem implementations in the kernel's
`fs/` directory - not bug fixes, feature flags, or VFS infrastructure changes.

Generated by diffing the `fs/` subdirectories across every mainline kernel
release from 0.01 to 6.17. Source data and tooling:
<https://github.com/bitplane/linux-source-import>

### Linux 0.01 (September 1991)

Minix filesystem only, implemented directly in `fs/` with no subdirectories
and no VFS layer. Linus developed Linux on Minix, so this was the natural
choice. 14-character filename limit, 64 MB max filesystem size.

* **minix** (read-write)

### Linux 0.95 (March 1992)

The VFS (Virtual Filesystem Switch) layer was introduced, allowing multiple
filesystem implementations. Minix moved to its own `fs/minix/` subdirectory.

### Linux 0.96c (1992)

* Added: **ext** - the first Linux-native filesystem (Remy Card). Removed the
  Minix limitations on filename length and filesystem size, but considered a
  stopgap.

### Linux 1.0 (March 1994)

The big bang. Eight new filesystems in one release:

* Added: **ext2** - replaced ext as the standard Linux filesystem (Remy Card,
  Theo Ts'o, Stephen Tweedie). Dominant for a decade.
* Added: **hpfs** - OS/2 High Performance File System (read-only)
* Added: **isofs** - ISO 9660 / High Sierra CD-ROM filesystem (read-only)
* Added: **msdos** - MS-DOS FAT filesystem
* Added: **nfs** - Network File System client
* Added: **proc** - process information pseudo-filesystem
* Added: **sysv** - System V / Coherent / Xenix filesystem
* Added: **xiafs** - another Minix replacement attempt (Frank Xia)

### Linux 1.1.35 (1994)

* Added: **umsdos** - Unix extensions on FAT (long filenames, permissions)

### Linux 1.3.7 (1995)

* Added: **smbfs** - SMB/CIFS network filesystem (Samba client)

### Linux 1.3.53 (1995)

* Added: **ncpfs** - Novell NetWare Core Protocol filesystem

### Linux 1.3.60 (1995)

* Added: **fat** - shared FAT support layer extracted from msdos
* Added: **vfat** - Windows 95 long filename FAT support

### Linux 1.3.94 (1996)

* Added: **affs** - Amiga Fast File System (OFS/FFS)
* Added: **ufs** - BSD Unix File System (read-only on Linux)

### Linux 2.1.21 (1997)

* Added: **romfs** - simple read-only filesystem for embedded use
* Removed: **ext** - the original Linux filesystem, superseded by ext2
* Removed: **xiafs** - lost the "Minix replacement" race to ext2

### Linux 2.1.70 (1997)

* Added: **coda** - distributed/caching filesystem (CMU)

### Linux 2.1.74 (1997)

* Added: **ntfs** - Windows NT filesystem (read-only, limited)

### Linux 2.1.78 (1997)

* Added: **hfs** - Apple Macintosh HFS (classic Mac)

### Linux 2.1.80 (1997)

* Added: **adfs** - Acorn Disc Filing System (RISC OS)

### Linux 2.1.120 (1998)

* Added: **qnx4** - QNX 4 filesystem

### Linux 2.3.2 (1999)

* Added: **efs** - SGI Extent File System (IRIX, read-only)

### Linux 2.3.17 (1999)

* Added: **udf** - Universal Disc Format (DVD/Blu-ray)

### Linux 2.3.25 (1999)

* Added: **bfs** - SCO UnixWare Boot File System

### Linux 2.3.35 (2000)

* Added: **cramfs** - compressed read-only filesystem for embedded

### Linux 2.4.0 (January 2001)

* Added: **jffs** - Journalling Flash File System (for NOR flash)
* Added: **ramfs** - simple RAM-backed filesystem

### Linux 2.4.x stable series (2001-2008)

Features backported across the 2.4 stable series. Canonical first appearances
are in the 2.5.x development series, noted below where applicable.

* Added: **ext3** - journalling ext2 (backported from 2.5.x)
* Added: **reiserfs** - journalling B*-tree filesystem (Hans Reiser)
* Added: **xfs** - SGI's journalling filesystem from IRIX
* Added: **jfs** - IBM's Journalled File System from OS/2 and AIX
* Added: **jffs2** - improved flash filesystem, replaced jffs
* Added: **befs** - BeOS filesystem (read-only)
* Added: **freevxfs** - Veritas VxFS (read-only)
* Added: **hfsplus** - Apple HFS+ (Mac OS 8.1+)

### Linux 2.5.6 (2002)

* Added: **jfs** (first mainline appearance)

### Linux 2.5.36 (2002)

* Added: **xfs** (first mainline appearance)

### Linux 2.5.42 (2002)

* Added: **cifs** - Common Internet File System (improved SMB client)

### Linux 2.5.43 (2002)

* Added: **afs** - Andrew File System (kAFS, in-kernel client)

### Linux 2.5.45 (2002)

* Added: **befs** (first mainline appearance)

### Linux 2.6.4 (March 2004)

* Added: **hfsplus** (first mainline appearance)

### Linux 2.6.7 (June 2004)

* Removed: **intermezzo** - distributed filesystem, abandoned

### Linux 2.6.13 (August 2005)

* Added: **9p** - Plan 9 filesystem protocol (v9fs)
* Added: **fuse** - Filesystem in Userspace

### Linux 2.6.14 (October 2005)

* Removed: **umsdos** - Unix-on-FAT, obsolete

### Linux 2.6.16 (March 2006)

* Added: **ocfs2** - Oracle Cluster File System 2

### Linux 2.6.18 (September 2006)

* Removed: **devfs** - device filesystem, replaced by udev/sysfs

### Linux 2.6.19 (November 2006)

* Added: **ext4** - fourth extended filesystem (started as ext3dev)
* Added: **gfs2** - Red Hat Global File System 2 (cluster)
* Added: **ecryptfs** - enterprise-level stacked cryptographic filesystem

### Linux 2.6.21 (April 2007)

* Removed: **jffs** - original flash filesystem, superseded by jffs2

### Linux 2.6.27 (October 2008)

* Added: **omfs** - Optimized MPEG Filesystem (ReplayTV/Karma)
* Added: **ubifs** - UBI File System (NAND flash, replaced jffs2)

### Linux 2.6.28 (December 2008)

* **msdos** and **vfat** merged into **fat/** - no longer separate directories

### Linux 2.6.29 (March 2009)

* Added: **btrfs** - B-tree filesystem (Chris Mason, Oracle)
* Added: **squashfs** - compressed read-only filesystem (Phillip Lougher).
  Existed out-of-tree since 2002, finally merged.

### Linux 2.6.30 (June 2009)

* Added: **nilfs2** - log-structured filesystem (NTT)

### Linux 2.6.34 (May 2010)

* Added: **ceph** - distributed filesystem (Sage Weil/Inktank)
* Added: **logfs** - log-structured flash filesystem

### Linux 2.6.37 (January 2011)

* Removed: **autofs** (v3) - replaced by autofs4
* Removed: **smbfs** - replaced by cifs

### Linux 3.4 (May 2012)

* Added: **qnx6** - QNX 6 filesystem

### Linux 3.8 (February 2013)

* Added: **f2fs** - Flash-Friendly File System (Samsung)

### Linux 3.18 (December 2014)

* Added: **overlayfs** - union/overlay mount filesystem

### Linux 4.2 (August 2015)

* Removed: **ext3** and **jbd** - folded into ext4 (ext4 mounts ext3 natively)

### Linux 4.6 (May 2016)

* Added: **orangefs** - parallel distributed filesystem (Omnibond, formerly PVFS2)

### Linux 4.10 (February 2017)

* Removed: **logfs** - never gained adoption, unmaintained

### Linux 4.16 (April 2018)

* Removed: **ncpfs** - Novell NetWare, obsolete

### Linux 5.1 (May 2019)

* Removed: **exofs** - object-store filesystem, hardware never materialized

### Linux 5.4 (November 2019)

* Added: **erofs** - Enhanced Read-Only File System (Huawei, for Android)

### Linux 5.6 (March 2020)

* Added: **vboxsf** - VirtualBox shared folders (guest driver)
* Added: **zonefs** - zoned block device filesystem (Western Digital)

### Linux 5.7 (May 2020)

* Added: **exfat** - Microsoft's FAT successor (Samsung contributed the driver)

### Linux 5.15 (October 2021)

* Added: **ntfs3** - full read-write NTFS driver (Paragon Software). The old
  read-only ntfs driver remained alongside.

### Linux 6.4 (June 2023)

* **cifs** and **ksmbd** merged into **smb/** - unified SMB client and server

### Linux 6.7 (January 2024)

* Added: **bcachefs** - copy-on-write filesystem with built-in caching,
  compression, encryption (Kent Overstreet)

### Linux 6.9 (May 2024)

* Removed: **ntfs** - old read-only driver, superseded by ntfs3

### Linux 6.13 (January 2025)

* Removed: **reiserfs** - long unmaintained, removed after extended deprecation

### Linux 6.15 (May 2025)

* Removed: **sysv** - System V/Coherent/Xenix filesystem, removed after 31
  years in-tree

### Linux 6.18+

Notable: bcachefs was removed in 6.18 following the maintainer's departure
from the project.
