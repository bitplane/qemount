# ITT 3030 disk image

The ITT 3030 was a Z80A-based business microcomputer sold by ITT (the German
arm, with development links to ITT/SEL) around 1979–1980. It ran a modified
CP/M held partly in on-board PROM and used 5.25-inch floppy drives.

This `.dsk` image is a raw, fixed-geometry sector dump of an ITT 3030 floppy.
In MAME it is built on the `wd177x_format` base class — the family of plain
MFM sector images decoded for Western Digital WD177x-class controllers — so it
carries no header or magic of its own; the bytes are simply the sector data in
track order.

## Structure

The loader recognises double-sided, double-density 5.25-inch geometry with two
track counts (matching the 40- and 80-track drive options documented for the
machine):

- 70 tracks or 35 tracks
- 2 heads (double-sided)
- 16 sectors per track
- 256 bytes per sector
- MFM encoding

The 70 × 2 × 16 × 256 layout gives the ~560 KB capacity the format is named
for. The MAME source notes its gap values are a guess, so low-level timing is
approximate. As a headerless sector dump it has no content signature to match
on, so no detection rule is documented here.
