# TI-99/4A floppy image (sector dump / track dump)

Disk images for the Texas Instruments TI-99/4A (and the compatible Geneve 9640).
MAME's `ti99_dsk` loader handles the two long-established emulator container
conventions for these disks: the **Sector Dump Format** (SDF, the "v9t9" form)
and the **Track Dump Format** (TDF, the "PC99" form). Both decode to the same
underlying TI disk; they differ only in whether the file stores logical sectors
or the full physical track stream.

## Sector dump format (v9t9 / SDF)

A flat concatenation of every logical sector, sector 0 first through the highest
sector number, with no per-track information and no clock/gap data. This is the
most common interchange form (extension `.dsk`). Standard geometries, all with
256-byte sectors:

- SSSD: 1 head, 40 tracks, 9 sectors → ~90 KB (FM)
- DSSD: 2 heads, 40 tracks, 9 sectors → ~180 KB
- DSDD: 2 heads, 40 tracks, 18 sectors → ~360 KB (MFM)
- 80-track / high-density variants up to ~720 KB+ (Geneve, HFDC)

MAME tolerates 40-track disks loaded in 80-track drives (double-stepping) and
strips an optional 768-byte bad-sector map appended to some images.

## Track dump format (PC99 / TDF)

Stores the raw track data — sector address marks, headers and inter-sector
gaps — but without the recovered clock patterns, so the file is roughly 50%
larger than the equivalent sector dump. This is the form used by the PC99
emulator; the suffix `.dtk` (and sometimes `.trk`) is conventional.

## Detection

The image containers themselves are headerless and identified by size and
geometry rather than a signature. The TI filesystem inside, however, has a
reliable on-disk marker: sector 0 holds the Volume Information Block (VIB),
whose bytes at offset 0x0D–0x0F contain the ASCII string `DSK`. Offset 0x0C
gives sectors-per-track (9 or 18) and offsets 0x11–0x13 give tracks-per-side,
sides and density. MAME and the TI-99 hardware documentation agree on this VIB
layout. Note this identifies the TI filesystem volume, not the SDF-vs-TDF
container; a track dump carries the same VIB embedded within its first track.

Related TI-99 media include the cartridge container `format/arc/rpk` (RPK, the
ROM-pack zip used by MAME for TI cartridges).
