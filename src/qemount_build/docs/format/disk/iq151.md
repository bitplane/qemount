# IQ 151 disk image

The IQ 151 was an 8-bit microcomputer built around an MHB8080A (an Intel 8080
clone) and produced by ZPA Nový Bor in Czechoslovakia from the mid-1980s,
mainly for use in schools. Storage was modular: the optional **DISC 2** module
drove an 8-inch floppy unit and ran an unlicensed CP/M-compatible operating
system.

`.iqd` is the raw sector-dump image used to represent those 8-inch floppies in
emulation. It is a headerless, fixed-geometry dump — there is no magic number
or descriptor block; the file is just the sector data laid out in track order
and identified by its extension and size.

## Structure

Per MAME's loader the geometry is a single-sided, single-density 8-inch disk:

- 77 tracks
- 1 head (single-sided)
- 26 sectors per track
- 128 bytes per sector
- FM/MFM low-level encoding via an i8272/uPD765-class controller

That 77 × 26 × 128-byte layout is the classic IBM 3740 single-density 8-inch
CP/M format, consistent with the DISC 2 module running a CP/M clone. The MAME
source flags its gap parameters as unverified, so exact low-level timing should
be treated as approximate.

Because the image is a plain sector dump with no header, there is nothing to
match on directly; it cannot be distinguished from other 8-inch SSSD CP/M dumps
by content alone, so no detection signature is documented here.
