# 7-Zip (.7z)

7-Zip was created by Igor Pavlov in 1999. It uses LZMA/LZMA2 compression
by default, achieving significantly better compression ratios than ZIP's
Deflate. The format and reference implementation are open source.

## Characteristics

- LZMA/LZMA2 compression (also supports PPMD, BZip2, Deflate)
- Solid compression (multiple files compressed as one stream)
- AES-256 encryption
- Large file support (up to 16 EiB)
- Unicode filenames
- Header compression

## Structure

```
Signature Header (32 bytes):
  Offset  Size  Field
  0       6     Signature (37 7A BC AF 27 1C)
  6       1     Major version
  7       1     Minor version
  8       4     Start header CRC
  12      8     Next header offset
  20      8     Next header size
  28      4     Next header CRC
```

The main metadata (file list, sizes, attributes) is stored in the
"next header" at the end of the archive, allowing streaming writes.

## Compression Methods

| ID   | Method   |
|------|----------|
| 0x03 | LZMA     |
| 0x21 | LZMA2    |
| 0x04 | PPMD     |
| 0x02 | BZip2    |
| 0x40 | Deflate  |
| 0x06 | AES-256  |
