# CP/68 disk image

A raw, headerless sector image for CP/68, a floppy-based operating system for the
Motorola 6800 published in 1979 by Jack E. Hemmenway and Robert D. Grappel. CP/68
is closely modelled on Digital Research's CP/M 1.4, adapted to the 6800. MAME's
loader reads a single-sided 5.25" disk of 35 tracks, 18 sectors of 128 bytes,
written in FM single density (~87.5 KB).

## Geometry

The image has no header; geometry is fixed:

| Capacity | Tracks | Sides | Sectors | Bytes/sector | Encoding |
|----------|--------|-------|---------|--------------|----------|
| ~87.5 KB | 35 | 1 | 18 | 128 | FM SD |

Track 0 holds boot sectors followed by directory sectors; the remaining tracks
hold file data. Sectors are physically interleaved. CP/68 boot disks begin with
6800 boot code (a `JSR`/`LDAA` sequence), which MAME's loader uses to
distinguish them, though this magic is described only by the MAME source and is
not corroborated independently here.
