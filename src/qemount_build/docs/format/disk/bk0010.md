# BK-0010 disk image

A raw, headerless MFM sector image for the floppy drives of the Soviet
Elektronika BK home-computer family. The BK series (BK-0010 from 1985, the
paged BK-0011 from 1989) were 16-bit machines built around the K1801VM1, a
Soviet LSI-11 / PDP-11 compatible CPU. The same MAME loader also serves the
related DVK and UKNC PDP-11-derived systems, which address the drives as the
`BY:`, `MY:` and `MZ:` devices respectively.

The image is decoded by MAME's Western Digital WD177x floppy framework, so it is
a straight dump of the disk's user data with no container header or filesystem
metadata in the file itself.

## Geometry

| Variant | Tracks | Sides | Sectors | Bytes/sector | Encoding | Media |
|---------|--------|-------|---------|--------------|----------|-------|
| Single-sided | 80 | 1 | 10 | 512 | MFM | 5.25" |
| Double-sided | 80 | 2 | 10 | 512 | MFM | 5.25" |

Rotation is 300 RPM. MAME selects the variant by file size; gap parameters are
taken from the BKBTL emulator.
