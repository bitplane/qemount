# ABC 1600 (Luxor floppy image)

A raw, headerless sector image for the Luxor ABC 1600, a 1985 Motorola 68000
workstation from DIAB/Luxor (Sweden) running the DNIX operating system. Its
5.25" drive stored 640 KB.

## Geometry

Fixed geometry, MFM encoding:

| Property | Value |
|----------|-------|
| Tracks | 80 |
| Sides | 2 |
| Sectors / track | 16 |
| Bytes / sector | 256 |
| Total | 640 KB (5.25" quad density) |

The image has no header; data is located by computing an offset from the track
and head numbers.
