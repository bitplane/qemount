# RC759 Piccoline floppy image

A raw, headerless sector image of a 5.25" floppy from the **Regnecentralen
RC759 "Piccoline"**, a Danish computer distributed from roughly 1984 to 1989 and
aimed at schools. The Piccoline used an Intel 80186 CPU and ran a Danish edition
of Concurrent CP/M-86; its floppies use a non-IBM layout, which is why a
dedicated image format exists rather than reusing a generic PC dump. The image
is a straight dump of the disk's sectors with no container header.

## Geometry

Per MAME's loader (derived from `wd177x_format`), the format is a fixed MFM
image:

| Property | Value |
|----------|-------|
| Form factor | 5.25" |
| Tracks (cylinders) | 77 |
| Heads | 2 |
| Sectors / track | 8 |
| Bytes / sector | 1024 |
| Sector numbering | starts at 1 |
| Total size | 1,261,568 bytes (~1.2 MB) |

The 77-cylinder, 1024-byte-sector layout is characteristic of CP/M-era
high-capacity floppies rather than the 80-cylinder PC norm. The format carries
no magic; it is recognised by geometry and the `.img` extension, so there is no
Detection section.
