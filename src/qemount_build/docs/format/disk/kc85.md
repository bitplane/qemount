# KC 85 disk image (Robotron)

A raw, headerless sector image of a 5.25" floppy for the Robotron KC 85, the
Z80/U880-based home-computer line built in the former East Germany (VEB
Mühlhausen). Floppy support came via the D004 expansion, a self-contained unit
with its own U880 CPU that drove the drives and let the machine run CP/M. The
image is a flat dump of sector data with no container header or per-track
metadata; MAME builds it on top of its generic uPD765 controller format.

## Geometry

The disks are 80-track, double-sided, double-density MFM, written at 300 RPM.
MAME's loader recognises three sector layouts, all 80/2:

| Sectors/track | Bytes/sector | Approx. capacity |
|---------------|--------------|------------------|
| 5 | 1024 | 800 KB |
| 9 | 512 | 720 KB |
| 16 | 256 | 640 KB |

The 80-track, double-sided, five-sectors-of-1024-bytes layout (~800 KB) is the
common Robotron CP/M format. The loader's source notes that the inter-sector gap
sizes are unverified. There is no signature; the format carries no header, so no
detection rule is given here.

This is the floppy side of the system; the KC 85 cassette encoding is catalogued
separately at [media/kc-cas](../media/kc-cas.md). Note that the Robotron
[A5105](a5105.md) is a different East German machine with its own disk format.
