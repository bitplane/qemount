# ISO 9660

ISO 9660 is the standard filesystem for CD-ROMs, published in 1988. It was
designed for read-only optical media with broad cross-platform compatibility.
Also known as CDFS (CD-ROM File System) or ECMA-119.

## Characteristics

- Read-only
- 8.3 filenames at Level 1 (DOS compatible)
- 31 character filenames at Level 2
- Maximum 8 directory levels (Level 1)
- 2048-byte logical blocks (matching CD sectors)
- Maximum file size: 4 GB (2^32 bytes)
- Maximum volume size: 8 TB

## Structure

- System Area: first 16 sectors (32KB), unused by ISO 9660
- Volume Descriptors: start at sector 16 (offset 0x8000)
- Primary Volume Descriptor has "CD001" at offset 0x8001
- Path Table: sorted directory listing
- Directory Records: file/folder metadata
- File data

## Extensions

| Extension        | Purpose                                 |
|------------------|-----------------------------------------|
| Rock Ridge       | POSIX permissions, symlinks, long names |
| Joliet           | Unicode filenames (Microsoft)           |
| El Torito        | Bootable CDs                            |
| Apple Extensions | Mac resource forks, Finder info         |

## Volume Descriptor Types

| Type | Offset 0x8000 | Description                    |
|------|---------------|--------------------------------|
| 0x00 | Boot Record   | El Torito boot catalog pointer |
| 0x01 | Primary       | Main volume descriptor         |
| 0x02 | Supplementary | Joliet/enhanced                |
| 0xFF | Terminator    | End of descriptors             |

## Use Cases

- CD-ROM and DVD-ROM distribution
- Live operating systems
- Software installation media
- Archival optical storage
