The RX50 is Digital Equipment Corporation's 5.25-inch floppy subsystem
introduced around 1982 for the DEC Rainbow 100, the DECmate II/III word
processors, and various PDP-11 / Professional 300 systems. DEC maximised
capacity on standard 5.25-inch media by recording 80 tracks on a single
side rather than the usual 40 tracks per side.

The geometry is 80 tracks, single-sided, 10 sectors per track of 512
bytes each, MFM (double density) at a 250 kbps data rate and 300 RPM.
That gives 80 x 10 x 512 = 409,600 bytes (400 KiB) per disk. MAME's
loader is labelled "DEC Rainbow 100 floppy image" and reserves the first
two tracks for loader / boot code. There is no header or magic: an image
is a raw sector dump identified by its 409,600-byte size, so no
signature detection is documented here. The MAME source leaves room for
unimplemented 40-track 9- and 8-sector variants.

Note that, although the low-level recording format is shared, the boot
sectors differ between machines: Rainbow diskettes carry Z80 boot code
that is not present on DECmate or PDP-11 disks.
