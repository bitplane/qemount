# Enterprise 64/128 disk image

A raw, headerless sector image of a floppy for the Enterprise 64 and Enterprise
128 home computers. The Enterprise (originally developed under the name Elan and
designed by the UK firm Intelligent Software) was a Z80-based machine launched in
the mid-1980s. Its disk system, EXDOS, drives a WD177x-family controller and
writes a standard IBM-compatible 3.5-inch double-density layout.

## Geometry

| Property | Value |
|----------|-------|
| Tracks | 80 |
| Sides | 2 |
| Sectors / track | 9 |
| Bytes / sector | 512 |
| Encoding | MFM (3.5" DSDD) |
| Total | 737,280 bytes (720 KB) |

The image carries no header and is a plain decoded sector dump in the usual
720 KB PC layout; MAME identifies it by extension and geometry rather than by any
magic bytes. (The MAME loader notes that its track gap sizes are unverified.)
