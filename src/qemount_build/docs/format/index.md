# Data Formats

Qemount works with various data formats for storing and organizing information
on disk. These are grouped into:

* [Filesystems](fs) - databases for storing files and directories
* [Partition Tables](pt) - schemes for dividing disks into regions
* [Disk Images](disk) - container formats for virtual disks
* [Archives](arc) - compressed collections of files
